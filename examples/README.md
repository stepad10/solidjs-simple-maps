# Examples

This directory contains reference implementations of common `solidjs-simple-maps` patterns.

## Available Examples

- [Basic Map](./BasicMap.tsx): A simple map rendering countries.
- [Zoomable Map](./ZoomableMap.tsx): A map with pan and zoom capabilities.
- [Map with Metadata](./MapWithMetadata.tsx): SEO-optimized map with OpenGraph and Twitter tags.

## Usage

You can copy these components directly into your SolidJS project. Ensure you have installed the necessary dependencies:

```bash
npm install solidjs-simple-maps solid-js @solidjs/meta
```
